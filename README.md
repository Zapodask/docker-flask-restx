# Docker flask-restx

## Instalação

```bash
# Criar ambiente virtual
$ python -m venv .venv

# Ativar ambiente virtual
$ .venv/Scripts/activate

# Instalar dependencias
$ pip install -r api/requirements.txt
```

Configurar arquivo .env seguindo o arquivo .env.example

## Iniciar

```bash
$ python start.py
```
